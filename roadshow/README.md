## 本地预览

在当前目录启动任意静态服务器，例如：

```powershell
python -m http.server 4182 --bind 127.0.0.1
```

然后访问 `http://127.0.0.1:4182/`。

页面支持鼠标滚动、右侧导航，以及方向键、Page Up、Page Down、Home 和 End 键翻页。
