# Styles

`src/style.css` 只负责按顺序导入样式模块。不要改变导入顺序，后面的关卡修饰层依赖现有 CSS 层叠关系。

```text
core.css                         全局变量、重置和基础布局
views/                           首页、地图、统一关卡页
components/                      跨页面小组件
levels/                          单个关卡专属样式
shared/                          多关共用的行动场景和视觉基线
```

维护规则：

1. 新关卡样式放进 `levels/`，使用关卡根类限定作用域。
2. 两个以上关卡真正复用的样式才进入 `shared/`。
3. 不把新规则继续堆回 `style.css`。
4. 不随意移动现有导入顺序；需要调整层叠时先做逐关视觉回归。
5. `_archive/` 中的旧样式仅作历史参考，不参与构建。
