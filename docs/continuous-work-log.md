# 持续工作记录

## 本轮范围

1. 统一七关任务提示和反馈系统。
2. 统一七关完成与历史复盘页面。
3. 从指定飞书文档整理七关图片史料并接入。
4. 实现七关差异化 AI 表达。

## 工作边界

- 不改变关卡玩法、判定、计时与胜负机制。
- 不删除现有内容；所有修改保留在功能分支并分阶段提交。
- 不调用付费生图，不推送、合并或部署。
- 不确定的史实不补写为正式结论。
- 安全、权限、付费、史实或产品方向问题记录后跳过，继续其他独立工作。

## 跳过事项

- 七张图片均通过当前可访问的飞书文档预览会话取得，没有绕过登录、权限或付费边界。
- 图片题名、作者、日期、馆藏与授权状态无法独立确认的部分统一保留为 `pending`，并在抽屉图注中标记“待审核”；没有把推测写成正式史实。
- 按约定未调用线上 MiMo、TTS 或生图接口，未产生付费请求；未推送、合并、部署或修改远端配置。

## 已完成

- 七关统一 Level Host 顶栏和五段式阶段显示。
- 七关统一右上角史料抽屉及空内容状态。
- 七关统一“本关问题”任务提示；顶栏自动显示关卡序号与既有玩家问题。
- 将各关原有提示归一为中性、成功、错误、辅助、史实确认五类反馈语义，保留原判定和计时逻辑。
- 七关共用“行动完成 · 历史复盘”组件：统一显示行动总结、时间地点和历史意义；有 AI 表达时与表达面板衔接，无表达时再领取碎片。
- 七关已启用同一 MiMo 接口下的差异化表达：分别生成出发札记、展品说明、会议记录、路线思考、行动电报、长征记忆卡和展台讲解；每关具备独立问题、表达角度、离线模板、结果标签和视觉色调。
- 从飞书《长征资源库》的“史料（图片）”区为七关各整理至少一张视觉史料，并接入右上角统一抽屉和 AI 史料选择；会宁保留原有四份文字史料并追加《通庄静会战役计划》图片。
- 本轮没有改变七关玩法、判定、计时与胜负机制；新增史料只进入统一资料层和表达层。

## 人工复核清单

- 核对遵义、四渡赤水、雪山草地三张手稿的准确题名、作者、形成日期与馆藏。
- 核对瑞金、湘江、泸定桥、会宁四张材料的馆藏著录及图片使用授权。
- 审核通过后，将对应 `rightsStatus` 从 `pending` 更新为 `reviewed` 或 `cleared`。
