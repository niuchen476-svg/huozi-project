# 长征档案行：样板关卡交互脚本

## 设计原则

本产品面向 K12 小学生，关卡不应让孩子被动背知识点，而应让孩子进入一个具体历史困境，通过操作、推理、反馈和收集，理解长征中的困难与转折。

每个关卡采用统一骨架：

1. 进入困境：用一段短场景让孩子知道“现在为什么危险”。
2. 介入操作：孩子通过拖拽、点击、摆放、排序、行动等方式参与。
3. 即时反馈：每一次操作都让画面、状态条、人物发言或记录纸发生变化。
4. 史料复盘：把刚才的操作与真实史料对应起来。
5. 获得碎片：通关后获得一件“长征记忆碎片”，进入最终汇合。

两个样板关的手感需要明显区分：

- 飞夺泸定桥：行动关。强调压迫感、速度、协同、冲桥动作。
- 遵义转折：推理关。强调线索整理、会议判断、因果关系和条理性。

## 样板关一：飞夺泸定桥

### 关卡定位

- 类型：行动调度 + 冲桥动作
- 玩家身份：红四团突击队协助员
- 核心问题：在时间极限、体力极限、桥面被拆、敌人火力封锁的情况下，怎样抢到泸定桥并打通北上通道？
- 通关碎片：铁索链环
- 目标体验：孩子玩完后明白，飞夺泸定桥不是“勇敢冲过去”这么简单，而是急行军、隐蔽、分工协同、火力掩护和突击精神共同完成的结果。

### 可用史料切入点

来自现有关卡数据：

- 5 月 28 日接到命令，必须 5 月 29 日清晨前夺取泸定桥。
- 剩余路程 240 里，限时不到 24 小时。
- 山路泥泞、天降暴雨，战士饥饿疲惫。
- 夜里敌军举火把赶路，红军也点火把伪装，隔河并行。
- 后半夜大雨浇灭火把，敌人休息，红军继续摸黑前进。
- 泸定桥桥板被拆，只剩 13 根铁索。
- 22 名勇士组成突击队，三连跟进铺桥板，6 挺重机枪压制敌人。
- 东桥头敌人点燃煤油和木板，勇士冲过火墙。

### 屏幕 0：任务下达

画面：

- 暗色战地图或当前已有 `pov-start.png` 背景。
- 屏幕中间出现军令纸。
- 背后有雨声、急促脚步声、远处枪声。

文案：

```text
1935 年 5 月 28 日夜，红四团接到命令：
必须在明天清晨前赶到泸定桥。
前方还有 240 里山路，追兵正在逼近。
```

玩家操作：

- 点击“接下任务”。

反馈：

- 军令纸折起，飞入左上角任务栏。
- 状态条出现：时间、体力、隐蔽。

知识点：

- 让孩子先知道“时间不够”是这一关的第一重困难。

### 屏幕 1：雨夜急行军

画面：

- 雨夜山路，队伍缓慢前进。
- 三个状态条：
  - 时间：距离天亮越来越近。
  - 体力：队伍越来越疲惫。
  - 隐蔽：被敌人发现的风险。

操作不要做成文字选择题，而做成可点击道具：

- 军号：催促队伍继续赶路。
- 干粮袋：短暂停下补充体力。
- 雨披/树影：让队伍隐蔽前进。

第一轮事件：

```text
战士们已经一天没好好吃饭，脚步越来越慢。
```

玩家可操作：

- 点击军号：时间保住，体力下降。
- 点击干粮袋：体力恢复，时间减少。

建议反馈：

- 点军号：
  ```text
  队伍咬牙继续向前。时间保住了，但体力下降。
  ```
- 点干粮袋：
  ```text
  战士们恢复了一点力气，但天色也更近了。
  ```

第二轮事件：

```text
河对岸出现敌军火把，敌人也在赶往泸定桥。
```

玩家操作：

- 把“火把”拖到队伍前方，伪装成同向行军。
- 或点击“熄火摸黑”，队伍减速但更隐蔽。

反馈：

- 火把拖对：敌军没有识破，隐蔽上升。
- 熄火摸黑：隐蔽上升，时间下降。
- 火把乱拖：暴露风险上升，出现“敌人似乎注意到了这边”。

第三轮事件：

```text
后半夜，大雨浇灭火把。敌人停下休息，红军还要不要继续走？
```

玩家操作：

- 点击“继续摸黑前进”。
- 点击“就地休息”。

设计建议：

- “继续摸黑前进”是最佳，但体力会很低。
- “就地休息”不会立刻失败，但会触发提示：如果停下，敌军增援可能先到桥头。

通过条件：

- 时间没有耗尽，隐蔽没有归零。

过场反馈：

```text
5 月 29 日清晨 6 点，红四团赶到泸定桥西岸。
机会，是用一整夜的急行军抢出来的。
```

### 屏幕 2：战前布阵

画面：

- 泸定桥简化战场图。
- 左侧三张单位卡：
  - 22 名突击队
  - 三连铺桥队
  - 6 挺重机枪
- 右侧三个部署点：
  - 铁索最前方
  - 突击队后方
  - 西岸火力点

玩家操作：

- 拖拽单位卡到对应位置。

正确对应：

- 22 名突击队 -> 铁索最前方
- 三连铺桥队 -> 突击队后方
- 6 挺重机枪 -> 西岸火力点

错误反馈要温和，不要像考试判错：

```text
机枪组不能冲在最前面。它们要留在西岸压制敌人，保护突击队前进。
```

```text
铺桥队要跟在突击队后方，这样后续队伍才能通过铁索桥。
```

全部放对后：

- 三条光线或箭头连起来。
- 出现“协同完成”印章。

总结文案：

```text
突击队夺桥，铺桥队跟进，机枪组压制火力。
这不是一个人的冲锋，而是一整支队伍的配合。
```

### 屏幕 3：铁索冲桥

画面：

- 使用现有 `pov-start.png`、`pov-mid.png`、`pov-fire.png`、`pov-arrival.png` 做阶段背景。
- 桥在晃，铁索横在画面中央。
- 底部显示队友头像和桥面进度。

操作设计：

- 空格或点击屏幕：向前攀爬。
- 鼠标点击对岸火力点：压制火力。
- 队友出现危险提示时，点击队友：提醒/保护队友。
- 火墙出现时，连续点击前进：冲过最后一段。

注意：

- 不建议做成“射击敌人身体”的玩法，面向小学生时可以表现为“压制火力点”。
- 目标可以是碉堡火光、枪口闪光、危险标记。

事件节奏：

1. 0%-30%：铁索摇晃，提示“抓紧铁索，保持前进”。
2. 30%-60%：对岸火力出现，玩家需要点击火力点压制。
3. 60%-85%：队友危险，玩家需要点击队友提醒。
4. 85%-100%：桥头火墙，玩家连续前进冲过。

失败条件：

- 不是“死亡失败”，而是“冲锋受阻，重新组织”。
- 失败后从冲桥段重来，不回到急行军。

成功反馈：

```text
两小时激战后，红军夺下泸定城。
大渡河上的北上通道，被打开了。
```

### 屏幕 4：史料复盘与碎片获得

画面：

- 桥上火光渐暗，出现一枚铁索链环。
- 链环旋转后飞入“长征记忆行囊”。

复盘卡片：

```text
你刚才经历的三件事，都能在史料中找到对应：

1. 240 里急行军：抢在敌军增援前抵达。
2. 三支队伍协同：突击队、铺桥队、机枪组共同夺桥。
3. 铁索冲桥：桥板被拆，只剩 13 根铁索。
```

碎片文案：

```text
获得记忆碎片：铁索链环
它见证了红军在没有退路时，打通北上道路的一刻。
```

## 样板关二：遵义转折

### 关卡定位

- 类型：会议推理 + 史料归类
- 玩家身份：会议旁听员 / 小记录员
- 核心问题：红军在损失惨重、被动挨打的危机中，如何判断问题根源，并找到新的方向？
- 通关碎片：会议记录纸
- 目标体验：孩子玩完后明白，遵义会议的“转折”不是一句口号，而是在危急时刻把问题、原因、决定理清楚。

### 可用史料切入点

来自现有关卡数据：

- 第五次反“围剿”失败后，红军被迫长征。
- 湘江战役后，中央红军从 8.6 万人锐减到约 3 万人。
- 惨重损失让将领们开始质疑原来的军事指挥。
- 1935 年 1 月 15 日至 17 日，遵义会议召开。
- 会议批评博古、李德军事指挥错误，否定死板阵地战。
- 增选毛泽东为中央政治局常委。
- 调整军事指挥，红军继续北上。
- 会议后，红军开始灵活机动，四渡赤水、巧渡金沙江、飞夺泸定桥。

### 屏幕 0：进入会议室

画面：

- 昏黄会议室，长桌、油灯、地图、茶杯、记录纸。
- 如果没有会议室插画，第一版可以用 CSS 做木桌背景和纸张。
- 玩家视角坐在桌边，身份是“小记录员”。

文案：

```text
1935 年 1 月，红军来到遵义。
湘江之后，队伍从 8.6 万人减少到约 3 万人。
这一次会议，要弄清楚：红军为什么陷入危险？接下来该怎么走？
```

玩家操作：

- 点击“开始记录”。

反馈：

- 空白会议记录纸摊开。
- 桌面散落多张史料卡。

### 屏幕 1：整理危机卡

目标：

- 让孩子先知道“发生了什么”。

桌上卡片：

- 湘江战役损失惨重
- 队伍从 8.6 万人减少到约 3 万人
- 敌军继续围追堵截
- 红军暂时在遵义获得休整机会

桌面区域：

- 危机事实
- 暂时机会

玩家操作：

- 将卡片拖到对应区域。

正确示例：

- 危机事实：湘江战役损失惨重、队伍锐减、敌军追击
- 暂时机会：攻占遵义，获得休整时间

反馈：

```text
你先把局面看清楚了：红军不是普通困难，而是已经到了非常危险的关头。
```

记录纸自动写入：

```text
一、当前危机：红军损失惨重，继续被敌军追击。
```

### 屏幕 2：找出原因卡

目标：

- 让孩子理解“问题不只是敌人强，而是指挥方式出了问题”。

桌上卡片：

- 死板阵地战
- 被动挨打
- 错误军事指挥
- 战士不勇敢
- 天气不好

桌面区域：

- 主要原因
- 干扰原因

玩家操作：

- 把卡片拖到区域。

设计重点：

- “战士不勇敢”是干扰卡。放错时提示：

```text
湘江战役中红军苦战五昼夜才突围。问题不是战士不勇敢，而是打法和指挥让队伍一直被动。
```

- “天气不好”也可以作为干扰卡。提示：

```text
天气会增加困难，但遵义会议重点讨论的是军事指挥和路线问题。
```

正确原因：

- 死板阵地战
- 被动挨打
- 错误军事指挥

记录纸自动写入：

```text
二、问题根源：错误军事指挥让红军长期被动。
```

### 屏幕 3：选择新方向

目标：

- 让孩子理解“转折”来自新的军事思路。

画面：

- 桌面出现两张大方案卡。

方案卡：

```text
继续正面硬拼
```

```text
灵活机动，寻找机会
```

玩家操作：

- 把一张方案卡拖到“会议建议”位置。

反馈：

- 选择继续硬拼：

```text
红军已经损失严重，如果继续用原来的硬拼办法，很可能再次陷入敌人的包围。
```

然后卡片弹回，让孩子重新尝试。

- 选择灵活机动：

```text
灵活机动可以避开敌人的优势。遵义会议之后，红军用四渡赤水、巧渡金沙江等行动重新争取主动。
```

记录纸自动写入：

```text
三、新方向：改变死板打法，采用灵活机动的运动战。
```

### 屏幕 4：完成会议记录

目标：

- 把遵义会议的核心决定整理成顺序。

桌上三张决定卡：

- 批评错误军事路线
- 调整军事领导
- 红军继续北上

玩家操作：

- 按顺序贴到会议记录纸上。

顺序建议：

1. 批评错误军事路线
2. 调整军事领导
3. 红军继续北上

反馈：

```text
会议记录整理完成。
这次会议让红军在危急关头重新找到方向。
```

### 屏幕 5：转折验证

画面：

- 会议桌上的地图亮起后续路线：
  - 四渡赤水
  - 巧渡金沙江
  - 飞夺泸定桥
  - 北上

文案：

```text
遵义会议之后，红军不再只是被动挨打，而是通过灵活机动不断争取主动。
这就是“转折”的含义。
```

### 屏幕 6：碎片获得

画面：

- 会议记录纸盖上红色印章。
- 纸张折成碎片，飞入“长征记忆行囊”。

碎片文案：

```text
获得记忆碎片：会议记录纸
它记录了红军在最危险的时候，重新找到方向的一刻。
```

## 两关之间的关系

遵义会议和飞夺泸定桥不应该是两个孤立小游戏。

可以用一句主线把它们连起来：

```text
遵义会议让红军找到新的方向；
飞夺泸定桥证明新的方向能在行动中打开道路。
```

地图上可以这样呈现：

- 通关遵义会议后，地图上的路线从暗色变为亮色，提示“红军重新获得主动”。
- 通关飞夺泸定桥后，大渡河节点亮起，提示“北上通道已打通”。

## 素材优先级

### 第一阶段：不等素材，先做可玩原型

可用 CSS 和现有图片完成：

- 泸定桥使用现有桥段图片。
- 遵义会议用 CSS 做木桌、纸张、卡片。
- 碎片用简单图标或文字徽章占位。

### 第二阶段：补充关键插画

优先补：

- 遵义会议室背景。
- 会议记录纸碎片。
- 铁索链环碎片。
- 三个单位图标：突击队、铺桥队、机枪组。

### 第三阶段：增强镜头和动画

可增加：

- 泸定桥桥面晃动、火光、雨夜行军镜头。
- 遵义会议卡片贴入记录纸的墨迹动画。
- 碎片飞入行囊动画。

### 暂不建议第一阶段做 3D

3D 展柜、3D 碎片拼合可以作为后期增强。当前最重要的是让两个样板关的操作、反馈、历史理解闭环成立。

---

## 下一步落地建议

优先实现顺序：

1. 遵义会议拖卡推理关。
2. 泸定桥急行军选择 + 布阵拖拽。
3. 将现有冲桥动作段接到布阵之后。
4. 增加“长征记忆碎片”收集状态。
5. 在地图页显示已获得碎片。

这样做的原因：

- 遵义会议几乎没有素材依赖，最适合先用 CSS 做出可玩版本。
- 泸定桥已有动作素材，只需要把前置调度补上。
- 碎片系统可以贯穿后续所有关卡，越早定下来越好。

## AI 互动设计：从“结尾点评”改成“关卡伙伴”

当前 AI 用法是：孩子写一段感悟，AI 点评并生成诗词。这个形式有教育意义，但可玩性偏弱，因为 AI 只在结尾出现，孩子在关卡中没有持续互动感。

更适合小学生的方式是把 AI 设计成“关卡里的伙伴”，但不要让 AI 直接给答案。AI 的角色应当是：

- 讲解员：把复杂史料说成孩子能懂的话。
- 小参谋：孩子卡住时给提示，不直接告诉正确答案。
- 复盘员：根据孩子刚才的操作，解释背后的历史意义。
- 提问者：用一两个问题引导孩子表达，而不是让孩子写很长感悟。

### AI 角色建议

可以给 AI 一个固定身份，例如：

```text
小小档案员
```

或：

```text
红军小参谋
```

它不代替孩子玩，也不判孩子“对错”，而是用短句陪孩子理解：

```text
你注意到了人数减少，这说明红军已经很危险了。
再想想：危险背后，是不是还有更深的原因？
```

### 遵义会议里的 AI 互动点

1. 动画导入后提问

```text
刚才你看到了什么？人数变少、追兵逼近、终于开会。
现在我们先把“发生了什么”整理出来。
```

2. 拖错卡片时提示

不要直接说“错了”，而是用启发式提示：

```text
这张卡说的是结果，不像原因。要不要放到别的地方试试？
```

3. 孩子犹豫时请求提示

界面可以有一个“问小参谋”按钮。点击后 AI 根据当前步骤给一句提示：

```text
看一看哪张卡和“为什么总是被动”有关。
```

4. 通关复盘

AI 不只生成诗，可以先给短复盘：

```text
你整理出的会议记录很清楚：先看见危机，再找到原因，最后选择新的方向。
这就是遵义会议为什么被叫作“转折”。
```

### 飞夺泸定桥里的 AI 互动点

1. 行军时做战况提醒

```text
天快亮了，敌人也在赶路。现在停下休息会更安全，还是会耽误时间？
```

2. 布阵时给战术提示

```text
突击队要冲在最前，机枪组的作用是保护他们，不是一起冲桥。
```

3. 冲桥后复盘

```text
你刚才做的不只是冲锋，还包括赶路、隐蔽和协同配合。
飞夺泸定桥的难，正难在这些事情必须同时完成。
```

### 接口形态建议

前端可以提供一个统一的 AI 助手接口：

```text
POST /api/assistant/hint
```

请求示例：

```json
{
  "levelId": "zunyi-turn",
  "scene": "cause-sorting",
  "childAction": "把“战士不勇敢”放进主要原因",

返回示例：

  "tone": "gentle-hint"
}
```

### 安全和教育边界

AI 给小学生用时要注意：

- 不输出过长内容，每次 1-3 句。
- 不使用恐吓、血腥、过度惨烈描述。
- 不替孩子直接完成任务。
- 不编造具体史实，如果不确定就说“我们回到史料卡看看”。
- 反馈以鼓励和引导为主。

### 分阶段实现

第一阶段不接真实 AI：

- 先写固定提示文案。
- 在界面上放“问小参谋”按钮。
- 让孩子体验“有人陪我推理”。

第二阶段接 AI：

- 根据关卡、步骤、孩子操作动态生成提示。
- 提示仍然限制在短句。

第三阶段增强个性化：

- 根据孩子犯错类型，生成不同提示。
- 根据孩子通关路径，生成个性化复盘。

## 遵义会议沉浸式素材占位方案

遵义会议关最终应从“UI 卡片拖拽”升级为“真实会议桌上的史料整理”。

### 场景结构

画面应像这样组织：

```text
昏黄会议室背景
  ↓
木质会议桌
  ↓
桌上散落：电报、战损报告、地图、发言纸条、铅笔、茶杯、油灯
  ↓
右侧或桌面中央：会议记录纸
```

孩子不是拖“卡片”，而是拖“史料物件”：

- 战损报告 -> 当前危机
- 追击地图 -> 当前危机
- 遵义会址/会议通知 -> 宝贵机会
- 错误指挥记录 -> 主要原因
- 灵活机动方案纸 -> 新方向
- 会议决定纸条 -> 会议记录
