# 七关 AI 表达矩阵

七关共用同一个 Level Host、表达面板、MiMo 服务、离线兜底和 TTS 朗读协议。差异只放在配置与结果卡视觉中，避免形成七套不可维护的 AI 代码。

| 关卡 | 输出形式 | 玩家要回答的问题 | 可选表达角度 |
|---|---|---|---|
| 瑞金出发 | 出发札记 | 离开熟悉的根据地时，最想留下什么 | 取舍物资、告别家园、保存力量 |
| 湘江血战 | 展品说明 | 怎样向观众说明突破背后的代价 | 时间代价、掩护队伍、突破并非轻松胜利 |
| 遵义转折 | 会议记录 | 这次讨论真正改变了什么 | 复盘指挥、改变方向、拿回主动 |
| 四渡赤水 | 路线思考 | 为什么改变路线也是主动 | 改变方向、迷惑追兵、创造机会 |
| 飞夺泸定桥 | 行动电报 | 极限时间里完成了什么、记住什么 | 抢时间、打开通道、突击与支援 |
| 雪山草地 | 长征记忆卡 | 从具体的人和事留下什么记忆 | 具体人物、彼此照顾、艰难取舍 |
| 会宁会师 | 展台讲解 | 怎样把史料与碎片讲成自己的理解 | 多路同向、重新会合、抵达也是出发 |

## 统一安全边界

- 模型只接收服务端配置、允许用于表达的史料摘要、玩家选择和玩家短文本。
- 玩家文本只作为表达素材，不能改变系统规则或输出格式。
- 在线生成失败自动使用本关专属模板，并明确显示离线状态。
- 在线结果使用本关专属标签，离线提示只在实际兜底时显示。
- 所有输出都标记为玩家表达，不冒充历史原文。
